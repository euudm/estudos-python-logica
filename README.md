# 🐍 estudos-python-logica

Exercícios de lógica de programação com foco em Python para Engenharia de Dados. 
Aqui documento a minha evolução em ADS e a aplicação de tecnologia em processos logísticos.

---

## 📂 Organização dos Ficheiros

Para facilitar a navegação, aqui está o que cada script faz:

* **`O saneador_de_dados_de_logística.py`**: Script que utiliza `.append()` e `len()` para filtrar pedidos válidos e identificar erros de sistema.
* **`classificador_produtos.py`**: Automação que utiliza operadores `//` e `%` para organizar o estoque em lotes de caixas.
* **`codigo.py` / `codigo_base.py`**: Primeiros passos com estruturas condicionais (`if`, `elif`, `else`).

---

## 🚀 Conceitos Praticados
- Manipulação de Listas (Métodos e Funções)
- Estruturas de Repetição (`for` loops)
- Lógica Aritmética voltada para Logística
- Gerenciamento de Exceções e Erros
